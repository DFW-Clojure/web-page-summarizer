# Web Page Summarizer

This app provides a gui for getting web pages and summarizing them. Demonstrates enlive and compojure.

## Prerequisites

You will need [Leiningen][1] 1.7.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein ring server
